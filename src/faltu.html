Problem Statement

The recruitment process is often delayed due to the time-consuming task of screening and shortlisting resumes manually. This not only increases the cost of hiring but may also lead to missing out on top talent. Human bias can affect candidate selection, resulting in potentially unfair hiring practices and missed opportunities for diversity.

Our Solution

SelectifyAI is an AI-powered resume screening and ranking platform designed to streamline recruitment by automating candidate evaluation. It supports batch processing of up to 100 resumes in formats like PDF and DOCX, and uses a hybrid method combining semantic and keyword analysis to achieve about 85% matching accuracy. The system ranks candidates based on relevance, offering data-driven shortlists and reducing manual effort.

Technology Stack

  • Natural Language Processing (NLP) using RoBERTa
  • Machine Learning algorithms for candidate ranking
  • Python backend with Flask API
  • React.js frontend with responsive design
  • MongoDB for resume data storage
  • Docker containerization for scalable deployment

Key Features

  • Automated resume parsing and information extraction
  • Customizable job requirement matching
  • Bias reduction through objective evaluation criteria
  • Candidate scoring and intelligent ranking system
  • Batch processing capability (up to 100 resumes)
  • Detailed analytics and recruitment insights

Methodology

Our platform implements a three-stage approach:
  1. Document Processing: Extract text from multiple document formats and normalize content
  2. Semantic Analysis: Use RoBERTa language model to understand context beyond keywords
  3. Intelligent Ranking: Apply weighted scoring algorithm based on job requirements

Benefits & Impact

  • 70% reduction in screening time
  • Improved quality of shortlisted candidates
  • Reduced unconscious bias in candidate selection
  • Data-driven recruitment decisions
  • Cost efficiency in hiring process
  • Scalable solution for organizations of all sizes
